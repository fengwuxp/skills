# 证据与安全边界

## 使用时机

当输出涉及经典原文、思想解释、现实类比、高风险决策或与其它专业 Skill 的责任冲突时读取。

## 不适用场景

- 只使用已核验事实且不引用经典的低风险决策卡。
- 只需调整普通措辞或格式。

## 读取后必须产出

- 标明哪些是现实事实、经典原文、现代解释和现实类比。
- 对未核验出处、关键事实和专业判断明确降级或停止。
- 给出适用边界、下一专业责任人和不得替代项。

## 需要继续读取的 reference

- 框架含义读 `classical-lenses.md`。
- 决策落地读 `decision-practice.md`。
- 已吸收来源和版本边界读 `source-map.md`。

## 四层证据

| 层级 | 内容 | 使用规则 |
| --- | --- | --- |
| 现实事实 | 用户材料、数据、合同、源码、日志、行为和确认 | 决策的首要依据；注明来源和时效 |
| 经典原文 | 已核对篇章、版本或可靠出处的原文 | 可以短引；未核对不得加引号或写“原话” |
| 现代解释 | 对经典概念的具名研究、通行解释或项目转译 | 明确是解释，不冒充原典唯一含义 |
| 现实类比 | 把经典镜片用于当前问题的假设 | 必须接受事实反证，不能证明现实结论 |

现实事实优先于经典解释。经典原文与现实事实冲突时，不调和事实；重新判断镜片是否适用。

## 引文纪律

- 引用前核对文本、篇章、版本或可靠来源；不同版本有差异时标注。
- 不拼接两处文本，不把俗语、后世概括或现代改写署名给古人。
- 不确定时改写为“可借此追问”，或交给 `hanzi-philology` 核验。
- 正式文档只保留必要短引和现实结论，不保留文化化推理过程。

## 高风险问题

- **医疗**：不诊断、不处方、不替代就医；出现高风险症状时优先建议及时寻求合格医疗帮助。
- **法律与合规**：不把经典义理当法律依据，核对当前法域和专业意见。
- **投资与重大财务**：不预测收益和运势，核对当前市场事实、损失承受能力和持牌意见。
- **安全与生产**：不替代测试、CR、审批、回滚和人工接管。
- **关系与权力**：不帮助欺骗、操控、胁迫、歧视、报复或规避监督。

## 专业责任冲突

| 用户真正需要 | 转交或协同能力 | 本 Skill 保留什么 |
| --- | --- | --- |
| 古籍、字源、异文、训诂 | `hanzi-philology` | 消费已核验结论做现实校准 |
| 产品与业务语义 | `product-architecture-expert` | 取舍、时势、反偏和止损问题 |
| 系分、代码、测试、CR、发布 | `senior-software-architect` | 决策前提与风险校准 |
| 正式报告和载体 | `document-authoring` | 已稳定的决策结论与证据等级 |
| 跨阶段、跨轮协作 | `wise-agent` | 专业镜片和决策卡 |

任何专业能力的事实结论优先于本 Skill 的类比；出现冲突时停止类比并回到原始证据。
