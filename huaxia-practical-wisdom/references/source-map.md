# 来源与吸收边界

## 使用时机

当需要核对华夏经世智慧的来源范围、外部 `huaxia-wisdom` 如何被使用，或判断某条内容能否进入本 Skill 时读取。

## 不适用场景

- 普通现实决策已经有充分事实，不需要引用经典或追溯来源。
- 需要古籍校勘、字源或训诂定论；此类任务交给 `hanzi-philology`。

## 读取后必须产出

- 使用了哪些来源、哪些只是现代解释或现实类比。
- 版本、篇章、时效和未核验边界。
- 可以吸收的方法与不得复制、不得外推的内容。

## 需要继续读取的 reference

- 经典框架的现实转译读 `classical-lenses.md`。
- 引文和高风险边界读 `evidence-boundaries.md`。

## 经典来源族

本 Skill 以这些来源族提供的稳定概念作为现代决策镜片：

- 《周易》与《易传》：变化、时位、阴阳关系。
- 《道德经》与《庄子》：知止、少干预、柔性和顺纹理。
- 《论语》《大学》《中庸》及后世儒家：正名、责任、中道、格物和知行。
- 《孙子兵法》与《韩非子》：风险、行动条件、规则和名实。
- 《黄帝内经》等中医经典：整体关系与治未病，仅作系统思考类比。
- 阴阳五行与五德终始相关传统：阶段承接、制约和退场，仅作生命周期类比。

本仓库不保存古籍全文，也不预设单一权威版本。任务需要直接引文、章句、异文或古义时，必须核对实际版本和篇章，必要时调用 `hanzi-philology`。

## 可以吸收

- 能转成事实问题、责任问题、风险问题、行动选择和验证条件的方法。
- 能帮助识别过度干预、名实分离、阶段错位、表象修补和不可逆风险的反偏问题。
- 经真实任务反复验证，且不依赖个人口味的最小场景路由。

## 不得吸收

- 古风人格、权威说教、固定口吻和角色扮演。
- 未核验名句、拼接引文、唯一思想解释和经典崇拜。
- 医学诊断、占卜命理、吉凶运势、宿命循环和权力合法性叙事。
- 个人偏好、私有对话、客户信息、生产数据、密钥或未经确认的现实判断。
- 把传统智慧写成事实、测试、CR、Git、审批或上线证据。
