# 音韵、借用与字际关系

## 使用时机

问题涉及通假、假借、声符、古今字、异体字、同源词、读音变化或以音求义时读取。

| 关系 | 判断重点 |
| --- | --- |
| 异体字 | 同一时代或相近系统中是否记录同一词、功能相同，仅形体不同 |
| 古今字 | 不同时代用字分工是否发生变化，后起字是否承接特定义项 |
| 通假 | 具体文句中是否因音近借用另一字，需语境、音韵和版本支持 |
| 假借 | 原字被借写另一词后是否形成稳定用法，不能只凭现代同音 |
| 声符关系 | 构形中的声符、时代音值和字族对应是否一致 |
| 同源词 | 音义对应和历史演变是否有系统证据，不能等同于同字 |

说明采用哪一种上古音、中古音或韵书体系；不同拟音体系并存时不要混用符号。音近只是候选条件，必须结合语法、语义、时代、异文和平行用例。字形相似、现代普通话同音或《说文》声训都不能单独建立通假关系。
