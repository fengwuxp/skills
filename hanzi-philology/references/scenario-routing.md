# 汉字学与训诂场景路由

## 使用时机

收到字源、古文字或古文解释问题后先读取，只选择与所问层次直接相关的材料。

| 问题 | 优先读取 | 必须补齐 | 不做 |
| --- | --- | --- | --- |
| 甲骨、金文、小篆字形演变 | `character-form-and-script.md` | 图版或器物编号、时代、释文来源、辞例 | 仅凭现代字形讲故事 |
| 本义、造字意图、形义关系 | `evidence-method.md`、字形与音韵 reference | 出土字形、同时代用例、音韵、不同释读 | 仅据《说文》定本义 |
| 通假、假借、古今字、异体 | `phonology-loan-and-variants.md` | 字际关系、音韵体系、语境和时代 | 只看字形相似或现代同音 |
| 《尔雅》《国语》或古文释义 | `exegesis-and-textual-context.md` | 原文上下文、底本、异文、传统注疏、平行用例 | 脱离篇章逐字释义 |
| 业务概念引用古文或存在训诂争议 | `evidence-method.md`、`exegesis-and-textual-context.md` | 原文、版本、语境、传统训释、现代用法和争议 | 替代产品 owner、用户研究或领域统一语言作最终命名 |
| 形成研究报告或 Word/PDF | 先完成训诂证据卡 | 结论等级、引用和不得扩写项 | 让文档工具重判证据 |
| API、类名、方法名、字段名和数据库命名 | 不适用 | 回到代码库、领域模型和项目约规 | 用训诂决定工程标识符 |
| 测字、起名吉凶、玄学 | 不适用 | 明确拒绝考据外扩 | 不包装成汉字学 |

研究对象、问题层次或材料范围不明确时，只问一个当前主 blocker。没有可核验来源时停止下定论。
