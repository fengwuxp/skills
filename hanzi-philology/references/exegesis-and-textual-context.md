# 训诂与文本语境

## 使用时机

解释《尔雅》《说文解字》《国语》、经史子集、注疏或短篇古文中的字词时读取。

1. 抄录最小必要原文和上下文，记录篇章、底本、版本及重要异文。
2. 先判断词在句中的语法位置、搭配和篇章作用，再比较同时代和平行用例。
3. 《尔雅》用于观察古代释词、同义归类和名物训释，《国语》用于观察具体叙事和语境用法；两者承担的证据角色不同。
4. 《说文解字》用于观察汉代字头系统、篆形和说解传统；遇到甲骨、金文、本义或上古用法问题，必须另找出土、音韵和辞例证据。
5. 历代注疏反映各时代的解释传统，应记录注家、时代和体例，不把后世解释无条件前推。
6. 文本存在异文或底本差异时，先列差异及其对释义的影响，再决定是否采用校改。

“小学”在本技能中指文字、音韵、训诂的综合方法，不是一部与《尔雅》《说文解字》《国语》并列的书。正式引用电子文本前，应核对其底本说明和可用影印本。
